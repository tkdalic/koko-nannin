# ここ何人？

日本全国の、地域ごとの人口を表示するサービス。

## demo url

https://tkdalic.github.io/koko-nannin/

## Requirements

言語: node v16

フレームワーク:  
Angular v13  
Angular Material v13  
Chart.js v2  
Storybook v6

## Install

1. `git clone https://github.com/tkdalic/koko-nannin.git`
2. `npm install`
3. `npm run start`
